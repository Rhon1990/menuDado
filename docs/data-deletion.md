# Solicitud de Eliminacion de Datos de MenuDado

Ultima actualizacion: 23 de junio de 2026

MenuDado no permite crear cuentas visibles dentro de la app. La app puede guardar datos localmente en el dispositivo y, cuando hay conexion, sincronizar algunos datos mediante Firebase con un identificador anonimo del dispositivo.

## Como eliminar datos locales

Puedes eliminar los datos locales de MenuDado de estas formas:

- Borrar menus desde la propia app.
- Borrar los datos de la app desde los ajustes de Android.
- Desinstalar MenuDado del dispositivo.

Esto elimina los datos guardados localmente en el dispositivo, como menus, preferencias del perfil alimentario, onboarding y contador local de uso de IA.

## Como solicitar eliminacion de datos asociados a la app

Para solicitar la eliminacion de datos asociados a MenuDado, escribe a:

rhonal.delgado@gmail.com

Incluye en el mensaje:

- Asunto: `Eliminar datos de MenuDado`.
- Nombre de la app: `MenuDado`.
- Pais desde el que usas la app, si quieres indicarlo.
- Modelo aproximado del dispositivo, si quieres indicarlo.
- Cualquier detalle que ayude a identificar la instalacion o el problema.

No envies contrasenas, documentos oficiales, datos financieros ni informacion medica sensible.

## Que datos se eliminaran

Cuando sea posible identificar los datos asociados a la solicitud, se eliminaran o desvincularan los datos sincronizados de MenuDado, como menus guardados, perfil alimentario, onboarding y uso diario de IA almacenados en Firebase.

Los datos agregados o anonimizados de analitica pueden conservarse cuando no permitan identificar directamente a una persona o una instalacion concreta, de acuerdo con las politicas de Firebase/Google y la configuracion del servicio.

## Plazo de respuesta

Las solicitudes se revisaran y responderan por correo electronico en un plazo razonable. Si hace falta informacion adicional para localizar los datos, se pedira por el mismo correo.
